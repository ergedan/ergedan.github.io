puppet流程控制语句

（1）if语句

格式：

if CONDITION {
        ...
}  elsif CONDTION {
        ...
} else{
        ...
}
CONDITION类型
  • 变量
  • 比较表达式
  • 有返回值的函数

示例：

  1 if $osfamily == 'Debian' {
  2         $apachename = 'apache2'
  3 }else{
  4         $apachename = 'httpd'
  5 }
  6 package{'$apachename':
  7         ensure  => latest,
  8 }

（2）case语句

格式：

case CONTROL_EXPRESSION {
    case1：{}
    case2：{}
    case3：{}
    ..
    default:{}
}
CONTROL_EXPRESSION
  • 变量
  • 表达式
  • 有返回值的函数
各case给定方式
  • 直接字串
  • 变量
  • 有返回值的函数
  • 正则表达式
  • default

示例：

  1 case $onfamily {
  2         "RedHat": { $webserver = 'httpd' }
  3         /(?i-mx:debin)}: { $webserver = 'apache2' }
  4         default: { $webserver = 'httpd' }
  5 }
  6 package{'$webserver':
  7         ensure  => latest,
  8 }

（4）selector语句

格式：

未知变量 = 可知变量 ? {
    case1   => value1,
    case2   => value2,
    ...
    default => value0,
}
CONTROL_VARIABLE的给定方法：
  • 变量
  • 有返回值的函数
各case的给定方式：
  • 直接字串；
  • 变量
  • 有返回值的函数
  • 正则表达式模式；
  • default

示例：

  1 $pkgname = $osfamily ? {
  2         "RedHat" => 'httpd',
  3         /(?i-mx:debian)/        => 'apache2',
  4         default => 'httpd',
  5 }
