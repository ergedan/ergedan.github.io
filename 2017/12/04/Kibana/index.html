一、Kibana

一个开源的分析和可视化平台。Kibana可以很容易地理解大量的数据，其简单的基于浏览器的界面使您能够快速创建和共享动态仪表盘，实时显示Eleasticsearch查询变化。

1、Kibana安装

下载地址：http://owatlfstl.bkt.clouddn.com/2017-12-07_174532.jpg

[root@node5 ~]# yum install -y kibana-6.0.0-x86_64.rpm

2、kibana.yml配置文件介绍

选项 说明
console.enabled 是否启用控制台
server.host 设置指定后端服务器的主机
server.name kibana示例的可读显示名称
server.port 指定端口
server.basePath 指定将Kibana挂载到的路径
elasticsearch.url 用于所有查询的Elasticsearch的示例的URL
elasticsearch.username和elasticsearch.password 使用Elasticsearch使用基于身份验证进行保护

3、启动Kibana

[root@node5 kibana]#vim kibana.yml

server.port: 5601
server.host: "172.18.4.67"
server.basePath: ""
server.name: "node5"
elasticsearch.url: "http://192.168.4.61:9200"

[root@node5 kibana]#systemctl start kibana

[root@node5 kibana]#systemctl status kibana
